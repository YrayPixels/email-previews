# Alertix Email Preview Files

This folder contains standalone HTML preview files for all Alertix alert email templates.

## Files

1. **01-adult-flag-alert.html** - Adult flag added/removed alert
2. **02-listing-status-alert.html** - Listing suppressed/inactive alert
3. **03-buy-box-alert.html** - Buy Box lost alert
4. **04-out-of-stock-alert.html** - Out of stock alert
5. **05-title-change-alert.html** - Product title changed alert
6. **06-features-change-alert.html** - Bullet points changed alert
7. **07-main-image-alert.html** - Main product image changed alert
8. **08-review-count-alert.html** - Review count changed alert
9. **09-number-of-offers-alert.html** - Number of offers changed alert
10. **10-multiple-alerts-example.html** - Example showing multiple alerts in one email

## How to Use

1. **Open in Browser**: Simply double-click any HTML file to open it in your default web browser
2. **Take Screenshots**: Use your browser's screenshot tools or system screenshot utility
3. **Test Responsiveness**: Resize your browser window to see mobile responsive behavior (< 600px width)
